<h3>How to install</h3>
<ul>
<li><h5> Install JBang <a href="https://github.com/jbangdev/jbang"> (github) </a></h5>

```bash

```

</li>


<li><h5> Run this </h5>

```bash
jbang NashornRepl.java
```

</li>
</ul>
